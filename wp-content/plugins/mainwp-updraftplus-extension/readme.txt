== Installation ==

1. Please install plugin "MainWP Dashboard" and active it before install MainWP UpdraftPlus Extension plugin (get the MainWP Dashboard plugin from url:http://mainwp.com/)
2. Upload the `mainwp-updraftplus-extension` folder to the `/wp-content/plugins/` directory
3. Activate the MainWP UpdraftPlus Extension plugin through the 'Plugins' menu in WordPress

== Screenshots ==
1. Enable or Disable extension on the "Extensions" page in the dashboard
